Let's make some worlds from rows!

The world is simple. It's made up of cells, each of which is On or Off.

Ok really, the world is composed of rows.

The cells in each row are determined by two things: (a) the previous row and (b) the set of rules we choose.

Each rule is encoded as a binary number between 0 and 255; this determines the values of the cells in the next row (On/Off).

Each cell in a new row (ignoring the end-cells) is adjacent to three cells in the previous row.

The values of those three cells (On/Off) determine the value of the new cell.
